
# Mushroom Classification using Decision Tree

This project applies a Decision Tree classifier to predict whether a mushroom is edible or poisonous using the UCI Mushroom Dataset.

## Dataset
UCI Machine Learning Repository:
https://archive.ics.uci.edu/dataset/73/mushroom

## Technologies
- Python
- Pandas
- NumPy
- Scikit-learn
- Seaborn
- Matplotlib

## Steps
1. Data preprocessing
2. One-hot encoding
3. Correlation analysis
4. Feature selection
5. Decision Tree training
6. Model evaluation

## Results
The model achieves high accuracy in classifying mushrooms.
